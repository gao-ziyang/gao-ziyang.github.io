# 代表项目

* **ZYNQ-7020 GEMM Accelerator**  
  基于 ZYNQ-7020 与 Vitis HLS 的 FPGA 加速器探索项目，关注指令级控制、共享 GEMM 调度，以及 CNN 和 Transformer 子模块在资源受限 FPGA 平台上的映射。

* **FPGA-based LMS Adaptive Filtering System**  
  面向未知电路模型探索的实时 FPGA 信号处理系统，集成激励生成、AD/DA 时序控制、LMS 自适应滤波和 MCU-FPGA 协同。

* **AI Titration Automation System**  
  基于计算机视觉、嵌入式控制和自定义硬件架构的自动滴定系统。我负责整体硬件架构设计，参与 SolidWorks 机械结构设计与 3D 打印部件制作，并将视觉终点检测与闭环注射控制集成到系统中。
