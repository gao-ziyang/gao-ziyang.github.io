# Selected Projects

* **ZYNQ-7020 GEMM / Attention Accelerator**  
  An FPGA-based accelerator project using Vitis HLS on ZYNQ-7020. The project explores instruction-level control, shared GEMM scheduling, and mapping of CNN and Transformer submodules such as Conv, QKV, and Attention.

* **FPGA-based LMS Adaptive Filtering and DDS System**  
  A real-time FPGA signal processing system for unknown circuit model exploration. The system integrates DDS excitation generation, AD/DA timing control, LMS adaptive filtering, and MCU-FPGA cooperation.

* **AI Titration Automation System**  
  An automated titration system based on computer vision and embedded control. The project uses a customized YOLOv8-based visual model and serial communication between the vision host and the microcontroller to support endpoint detection and closed-loop injection control.
